WORKER_MANAGER_TASK
	A simple web application which have 2 roles for users (manager, worker)
		[Manager] can create / edit / assign / reward task.
		[User] can view / upload / complete / search a task, view his total reward points.
		Application have search and filter options to view available tasks
	
APP - Angular
API - Node
DB - MongoDB

Prerequiste:
	Node stable version - https://nodejs.org/en/download/
	MongoDB - https://www.mongodb.com/try/download/community
	Install node modules - npm i
	Start the app (both App and Api) - npm start
